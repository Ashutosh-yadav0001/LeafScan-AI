# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-03-01

### Changed
- Rewrote LaTeX project report with simplified references
- Updated bibliography to include only beginner-friendly sources

## [1.0.0] - 2026-02-28

### Added
- Full-stack plant disease detection app (Potato + Tomato)
- CNN models trained on PlantVillage dataset
  - Potato model: 3 classes, 97.2% accuracy
  - Tomato model: 10 classes, 89.6% accuracy
- FastAPI backend with dual-model support
- React frontend with drag-and-drop upload and prediction history
- Docker & Docker Compose support
- Comprehensive documentation (README, GPU setup notes, project report)
- Sample test images
